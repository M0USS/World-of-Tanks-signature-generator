WoT Signature generator
Written by SkaceKachna (zipek.cz,menxmenx@gmail.com)
========================
Minimal requirements:
========================
  PHP 5+
  GD Library
  CUrl
  mod_rewrite (for .png adress. Not really needed)
  
========================
Installation:
========================
1) Unpack all files from archive (or both archives) to folder, where you want to have signature
2) Run wot_settings.php on server. Configure your signature.
3) Delete wot_settings.php, so noone can reconfigure your settings.
4) If you enabled caching of image, you'll may need to set "cache" and "wot_data" chmod to 777 
5) Done.
